# AI SEO positioning

## User request

Update Learn Word public positioning because the project is no longer just a learning app: it now has useful AI features. Improve in-app descriptions and SEO optimization.

## Scope

- Public frontend copy on Home and About pages.
- Static document SEO metadata.
- Static crawler assets for public pages.
- Frontend behavior spec.

## Acceptance criteria

- Public copy describes Learn Word as an AI-assisted vocabulary learning app.
- SEO metadata mentions AI-generated draft cards, collections, translations, transcriptions, and spaced repetition.
- Keywords include English and Russian terms for Google and Yandex discovery.
- `robots.txt` and `sitemap.xml` are included in the Angular build.
- Docker-based frontend build and local startup verification pass.

## Out of scope

- Backend API behavior changes.
- New AI generation features.
- Authenticated user-flow changes.
